## Install Packages

`go install`

## Generate Ent related files

`go generate ./...`

## Install ngrok

`brew install ngrok/ngrok/ngrok`

## Install Redis

`brew install redis`

## Install PostgreSQL

`brew install postgresql`

## Redis startup

`brew services start redis`

## Run Postgres

`brew services start postgresql`

## Run the server

`go run ./cmd/gameon`
